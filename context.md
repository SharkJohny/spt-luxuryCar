# Kontext projektu

## Technologie

- Shoptet Bender pro lokální vývoj
- Node.js projekt
- MCP konfigurace (zatím nefunkční)

## Struktura projektu

- `/finalHTML` - HTML soubory
- `/assets` - statické soubory (js, css, obrázky)
- `/dist` - zkompilované soubory

## Stav

- Bender je nainstalován
- Čekáme na URL e-shopu pro spuštění Benderu
- MCP konfigurace vyžaduje úpravu

## Poznámky

- Projekt používá esbuild pro bundling JavaScriptu
- MCP konfigurace v mcp.json používá neexistující balíček browsertools-mcp
